# Flawless Video Bot

Telegram-бот для скачивания видео из TikTok, Instagram Reels и YouTube Shorts без водяных знаков.

### Деплой на Render.com
- Build Command: `pip install -r requirements.txt`
- Start Command: `python bot.py`

### Переменные окружения
- `BOT_TOKEN` — токен от @BotFather
- `CHANNEL_ID` — ID твоего канала (например, -100XXXXXXXXXX)
- `CHANNEL_USERNAME` — @username канала (опционально)

Бот требует подписку на канал для использования.
